# Piko Assignment Marketplace

A simple static web app for selling assignments and study projects.

## Features
- Browse ready-made assignments by subject, level, and deadline
- Add assignments to a shopping cart
- View total price and checkout prompt
- Responsive layout for desktop and mobile

## Files
- `index.html` — main app page
- `piko.css` — styling for the marketplace
- `piko.js` — product listing, filter, and cart logic

## Run locally
Open `index.html` in your browser to use the app.
